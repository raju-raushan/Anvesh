# TVISHI 2.0 Futuristic Hackathon Website

A premium, futuristic, and fully responsive hackathon website for **TVISHI 2.0** by **IEEE Student Branch SAGE University Indore**.

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- React Icons
- Three.js via React Three Fiber + Drei
- EmailJS for contact delivery

## Features

- Futuristic animated loader
- Custom interactive cursor
- 3D hero background with rotating object + stars
- Smooth parallax and section transitions
- Animated stats, tracks tilt cards, glowing timeline and prize cards
- Expandable rules cards
- Memories gallery hover animations
- Register CTA and SMTP-style contact form integration
- Mobile/tablet/desktop responsive layout

## Run

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and set:

```bash
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```
