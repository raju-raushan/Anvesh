# TVISHI 2.0 Hackathon Website

Modern responsive single-page website for **TVISHI 2.0** by **IEEE Student Branch SAGE University Indore**, built with React + Vite, Tailwind CSS, Framer Motion, and React Icons.

## Setup

```bash
npm install
npm run dev
```

## Email Setup (SMTP via EmailJS)

1. Copy `.env.example` to `.env`.
2. Add your EmailJS service, template, and public key values.
3. Use those IDs inside the EmailJS dashboard template fields.

## Build

```bash
npm run build
npm run preview
```
